# m-chess
A CLI multiplayer chess game

## Run
```
$ python3 main.py
```

## Requirements
Opolo :mind-blown:

## Usage
   - player 1 is white so player to is......indian :joke:
   - to move pieces
     - pb3 (pawn to b3). -For both players
   - "Q." represents player 2 (pieces with a period)
   - "B " represents player 1 (pieces white whitespace)
